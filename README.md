# rallax

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

Parallax library for SolidJS

## Quick start

Install it:

```bash
npm i rallax
# or
yarn add rallax
# or
pnpm add rallax
```

Use it:

```tsx
import rallax from "rallax"
```
